# vue 2 + tailwind

Vue 2 CLI Starter with tailwind 2 and removed useless boilercode

## Added
- tailwindcss - 2.0.2
- @tailwindcss/forms - 0.2.1

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
